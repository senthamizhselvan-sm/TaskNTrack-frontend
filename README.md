# TaskNTrack - Client

React + Vite frontend using Tailwind and Axios.

Setup

1. cd client
2. npm install
3. Copy `.env.example` to `.env` and adjust `VITE_API_URL` (e.g., `http://localhost:5000`)
4. npm run dev

Notes

- Tailwind is configured via `tailwind.config.cjs` and `postcss.config.cjs`.
- The app expects the API at `VITE_API_URL`.
